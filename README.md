
# 🦷 Dentist Clinic Management System

A full-featured multilingual web platform for a dental clinic, complete with an admin dashboard for managing all clinic content in **Arabic**, **Turkish**, and **English**.

---

## 🇹🇷 Türkçe

### 🦷 Diş Kliniği Yönetim Sistemi

Diş kliniği için geliştirilmiş çok dilli bir web platformudur. Admin paneli üzerinden klinik içerikleri **Arapça**, **Türkçe** ve **İngilizce** olarak yönetilebilir.

---

## ✨ Features / Özellikler

### 🔧 Admin Dashboard / Yönetici Paneli

**🇬🇧 English**
- **Before & After Cases** – Add or delete patient treatment cases with before-and-after images.
- **Social Media Videos** – Add videos from social media with a custom thumbnail and link; view or remove them from the dashboard.
- **Services** – Add clinic services in three languages (Arabic, Turkish, English) with ready-made icons.
- **Doctors** – Add doctors with their CVs; fully manageable in all three languages.
- **Multilingual Content** – All entries (cases, services, doctors) are managed in **Arabic**, **Turkish**, and **English**.

**🇹🇷 Türkçe**
- **Önce & Sonra Vakaları** – Hasta tedavi vakalarını önce-sonra görsellerle ekleyin veya silin.
- **Sosyal Medya Videoları** – Sosyal medyadan videolar ekleyin; özel küçük resim ve bağlantı ile panelden görüntüleyin veya kaldırın.
- **Hizmetler** – Kliniğin sunduğu hizmetleri hazır ikonlarla üç dilde ekleyin.
- **Doktorlar** – Doktorları özgeçmişleriyle ekleyin; tüm bilgiler üç dilde yönetilebilir.
- **Çok Dilli İçerik** – Tüm girişler **Arapça**, **Türkçe** ve **İngilizce** olarak yönetilir.

---

### 🌐 Public Website / Genel Web Sitesi

**🇬🇧 English**
- Multilingual interface for visitors (Arabic, Turkish, English).
- **WhatsApp Contact** – Users can fill in their information on the main page and send a message directly via WhatsApp. The message is delivered in a structured, automated format.

**🇹🇷 Türkçe**
- Ziyaretçiler için çok dilli arayüz (Arapça, Türkçe, İngilizce).
- **WhatsApp İletişim** – Kullanıcılar ana sayfada bilgilerini doldurarak doğrudan WhatsApp üzerinden mesaj gönderebilir. Mesaj düzenli ve otomatik bir biçimde iletilir.
